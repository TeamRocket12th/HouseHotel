# House-Hotel 旅館預約服務

火箭隊 12 梯前端組協作作業

### 功能
* 可以個別觀看 6 種房型的詳細資訊
* 用日曆方式，瀏覽未來 90 天已預約與尚未預約的時段
* 在選擇預約日期時，會即時顯示訂房價格總價與天數
* 在線上訂房，只能預約未來 90 天內的時段，若已被預定則無法選擇
* 預約成功，會回傳成功通知 
* 預約失敗，會回傳訊息讓客戶得知預約未完成
